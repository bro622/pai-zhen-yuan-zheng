# 牌阵远征

回合制明牌策略卡牌游戏。预判 Boss 意图，规划能量，闯过五层试炼。

## 玩法

- **4x4 卡牌网格** — 8 种卡牌各有独特效果
- **能量管理** — 每回合有限能量，攻击/技能/防御需要权衡
- **Boss 意图** — 每回合预览 Boss 下一步行动，提前叠护甲或全力输出
- **5 层地图** — 分支路线选择，战斗/休息/最终 Boss

## 卡牌一览

| 卡牌 | 消耗 | 效果 |
|------|------|------|
| 龙 | 2 | 造成 14 伤害，自身 -3 生命 |
| 凤 | 1 | 恢复 3 生命 [消耗] |
| 虎 | 2 | 力量 +2 [消耗] |
| 鹤 | 0 | 能量 +1 [消耗] |
| 莲 | 1 | 对 Boss 施加 1 层虚弱 |
| 竹 | 1 | 获得 5 护甲 |
| 剑 | 1 | 造成 6 伤害 |
| 云 | 1 | 造成 3 伤害，施加 1 层易伤 |

## 技术栈

- **语言**: C++17
- **图形库**: SFML 3.0.2
- **构建**: CMake + Ninja
- **音效**: 25 种战斗/UI 音效
- **美术**: AI 生成，透明背景

## 构建

```bash
# 需要 SFML 3.0.2 安装在 C:/SFML-3.0.2
mkdir build && cd build
cmake .. -G Ninja
ninja
```

运行 `PaiZhenYuanZheng.exe`。

## 操作

| 按键 | 功能 |
|------|------|
| 鼠标左键 | 点击卡牌/按钮/节点 |
| Enter / 空格 | 确认 / 继续 |
| 1 / 2 / 3 | 奖励选择 |
| H / ? | 帮助面板 |
| R | 游戏结束后重来 |
| Esc | 退出 |
